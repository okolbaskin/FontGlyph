## FontGlyph

FontGlyph is a lightweight Windows application for viewing and analyzing TrueType glyphs. 
It is written in C++ using WinAPI and includes custom geometry and rendering utilities.

### Features

- View glyph outlines from any installed TrueType font

- Zoom and pan for detailed inspection

- Pure C++ implementation without external libraries

- Custom UI controls and drawing routines

- Fast rendering using custom geometric primitives

### Build

Open **FontGlyph.sln** in Visual Studio and build the project.\
The application runs on Windows 10 or later.
